# Smart Translator 🌍

A modern, fast, and responsive web translator built with vanilla JavaScript and SCSS, featuring real-time translation and a sleek modular UI.

## ✨ Features

- **Real-time Translation:** Powered by the MyMemory API.
- **Auto Language Detection:** Detects user's browser language for an instant localized experience (UA/EN support).
- **Dark Mode:** Stylish theme switching with persistent storage via `LocalStorage`.
- **Translation History:** Keep track of your past translations with a convenient sidebar.
- **Copy to Clipboard:** One-click copying with interactive feedback.
- **Modular Architecture:** Built using ES6 Modules for clean and maintainable code.
- **Modern Icons:** Integrated with [Phosphor Icons](https://phosphoricons.com/) for a professional look.
- **Advanced SCSS Architecture:** Modular styles with custom mixins for flexbox and positioning.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure with custom `data-i18n` attributes.
- **SCSS:** Professional modular structure (\_variables, \_mixins, \_animations) with a compiler-ready setup.
- **JavaScript (ES6+):** - Async/Await for API calls.
  - ES6 Modules for clean logic separation (`import/export`).
  - `navigator.language` for i18n.
  - `document.documentElement` for theme management.
  - LocalStorage for theme and history persistence.

## 🚀 Development & Build

This project uses **Sass (SCSS)** for styling. To manage dependencies and automate the styling workflow, `npm` is used.

1. **Install Dependencies:**
   Ensure you have [Node.js](https://nodejs.org/) installed, then run:
   ```bash
   npm install
   ```
2. **Development Mode (Watch):**
   To automatically compile your .scss files whenever you make changes:
   ```bash
   npm run watch
   ```
3. **Production Build:**
   To generate a minified and optimized CSS file before deployment:
   ```bash
   npm run build
   ```
4. **Local Execution:**

    Since this project utilizes ES6 Modules, you must serve index.html through a local server (e.g., VS Code Live Server extension) to avoid CORS issues.
